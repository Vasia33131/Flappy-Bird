О проекте
Жанр: Arcade / Hyper-Casual
Движок: Unity
Язык: C#
Платформа: WebGL / PC / Mobile

Управление
Пробел / ЛКМ / тап по экрану — взмах крыльями (птица поднимается вверх)

Бездействие — птица падает вниз под действием гравитации

Механика
Птица постоянно падает (симуляция гравитации)

Каждый тап даёт импульс вверх

Нужно пролетать между трубами, не задевая их

За каждую успешно пройденную трубу даётся 1 очко

При столкновении с трубой, землёй или потолком игра заканчивается

Типы препятствий
Препятствие	Описание
🟢 Обычные трубы	Классические зелёные трубы с разрывом посередине
